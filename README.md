## ✨ 核心特性：四大降维打击

### 🎭 1. 专属数字生命引擎 (零感定妆系统)
彻底告别复杂的图库匹配与路径玄学。
* **全局人脸锚点**：在 WebUI 中一键上传专属助理参考图（支持 `png/jpg/webp`）。插件将在 AstrBot 专属的隔离沙盒 (`plugin_data`) 中为您永久锁死这张“定妆照”。
* **极速秒级挂载**：无论重启多少次，每次触发自拍时，底层将以 0 延迟极速提取物理级绝对路径，**100% 命中，永不报错**。
* **人设自洽**：支持配置独立的“外貌特征 Prompt”（如 *1girl, masterpiece, silver hair*），系统会自动与用户的动作描述完美融合。

### 👁️ 2. 业界首创：双轨视觉控制流 (Dual-Track Vision)
打破了“固定人设”与“动态要求”不可兼得的痛点。插件在后台隐秘运行着两套视觉捕获雷达：
* **主轨 (Persona Ref)**：抓取 WebUI 中的全局人设图，确保“脸还是那张脸”。
* **副轨 (User Ref)**：实时监听聊天窗口。当您发送一张衣服/姿势的图片，并对 AI 说“穿这件衣服自拍”或“/画 这个姿势”时，系统会**瞬间捕获当前消息帧的图片**。
* **双图同传融合**：底层接口将两张图片同时送入大模型，实现**“固定形象 + 动态姿势/服装”**的完美合体！

### 🤖 3. 深度 LLM 泛化赋能 (真正的智能画师)
内置两个高权重的智能 Agent 工具 (`generate_selfie` & `generate_image`)，无需死板的斜杠命令，自然语言即可唤醒：
* **智能自拍扩写**：当您在群里说“看看腿”、“发张自拍”，LLM 会自动调用自拍工具，并**根据上下文语境、天气、情绪**，自动撰写极具细节的高质量英文 Prompt（包含服装、光影、镜头角度），随后无缝移交画图节点。
* **无感多模态交互**：结合动态捕获技术，您可以直接发一张图说“把这个画成赛博朋克风”，AI 会直接理解并执行，无需任何多余参数。

### 📡 4. 满级防弹底层架构 (Bulletproof API)
历经无数次报错洗礼，重构出的“三维立体防漏” API 路由系统：
* **严格遵循底层协议**：针对标准 OpenAI 兼容模型，智能判断是否有参考图。无图则走轻量级 `/generations` JSON 格式；有图则**自动切轨至 `/edits`**，并硬核构建 `multipart/form-data` 二进制表单直传，彻底解决参考图被网关吞噬的行业痛点。
* **多模态大模型原生支持 (Chat Vision)**：如果您使用的是支持视觉的对话大模型，插件将智能切轨至 `/chat/completions`。通过极致优化的 Base64 实时转码引擎，将“人设图”与“姿势图”作为 `image_url` 数组双重注入，完美榨干多模态大模型的视觉理解能力。
* **超强纠错导入**：无视框架层面的模块路径错乱与扁平化陷阱，采用三级降维打击的绝对路径导入机制，彻底消灭 `ModuleNotFoundError`。

## 💻 极简优雅的工作流

仅仅需要三步，即可让你的 AI 助理拥有最强视觉能力：

1.  **一键绑定**：在 WebUI 中填入 API 密钥，点击上传一张老婆/助理的照片，保存。
2.  **日常召唤**：
    * 在群内随意聊天：“今天天气真好，发张你逛街的照片看看。”
    * *(AI 思考中... 提取人设图 -> 扩写逛街 Prompt -> 并发请求)*
    * **出图！**
3.  **高级换装/模仿**：
    * 发送一张 JK 制服的图片。
    * 附带文字：“/自拍 穿这套衣服看着镜头”。
    * *(系统同时捕获 WebUI 人脸图 + 聊天窗口衣服图 -> 完美融合)*
    * **出图！**

## 🧠 技术栈与架构亮点 (For Geeks)

* **完全异步化 (Aiohttp)**：底层全量采用非阻塞协程，即使生成 8K 级高清大图，也绝不阻塞机器人主进程的消息处理。
* **智能兜底机制 (Fallback)**：图片路径容灾处理。从 `plugin_data` 绝对路径 -> 公共 `data` 降级 -> 全盘 `os.walk` -> URL 直读，确保文件寻址万无一失。
* **参数自动净化**：对接第三方（如 Banana）接口时，自动清洗并重组 Payload，规避低级网关因为不识别多余参数而报错的“暗坑”。
